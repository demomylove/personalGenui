社区 (Community)

欢迎来到 A2UI 社区！我们正在共同构建代理驱动界面 (Agent-Driven Interfaces) 的未来。

参与其中

A2UI 是一个基于 Apache 2.0 许可的开源项目。我们欢迎开发者、研究人员以及任何对推动代理式用户界面感兴趣的人士做出贡献。

项目合作伙伴

A2UI 与以下多个组织合作开发：

Google Opal

Opal 允许用户使用自然语言构建、编辑和分享 AI 迷你应用。Opal 团队一直是 A2UI 开发的核心贡献者。

Flutter

Flutter 的 GenUI SDK 使用 A2UI 作为 UI 声明格式，用于在 Flutter 应用程序中生成动态、个性化的 UI。

Gemini Enterprise

A2UI 正在被集成到 Gemini Enterprise 中，以使自定义代理能够在企业应用程序中渲染丰富、交互式的 UI。

AG UI / CopilotKit

AG UI 和 CopilotKit 提供了与 A2UI 的首日兼容性，使开发者能够构建状态同步的富应用程序，并从代理渲染动态 UI。

A2A

Google 的 A2A 团队 一直是 A2UI 开发的核心贡献者，并得到了 A2A 技术指导委员会 (TSC) 的支持。

行为准则 (Code of Conduct)

我们致力于为每个人提供一个受欢迎和包容的环境。所有参与者都应做到：

  • 尊重和体谅他人
  • 欢迎新人并帮助他们入门
  • 关注对社区最有益的事情
  • 对他人表现出同理心

绝不容忍不可接受的行为。请向项目维护者报告疑虑。

认可 (Recognition)

我们感谢所有的贡献！贡献者将在以下位置获得认可：

  • GitHub 的贡献者图表
  • 发布说明 (Release notes)
  • 项目致谢

重大贡献者可能会被邀请加入项目的维护者团队。

保持更新

  • Watch GitHub 仓库 以获取通知
  • Star 仓库 以收藏并表示支持
  • 关注 Release 以获取新版本的通知
